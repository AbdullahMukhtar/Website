# SEO Setup Guide - External Tasks

This guide covers the external SEO tasks that need to be completed manually (not in code).

## 1. Google Search Console Setup (CRITICAL - Do This First!)

### Steps:
1. Go to https://search.google.com/search-console
2. Click "Add Property" and enter: `verixaservices.com`
3. Verify ownership using one of these methods:
   - **DNS TXT Record** (Recommended):
     - Google will provide a TXT record like: `google-site-verification=abc123...`
     - Add this to your DNS settings at your domain registrar
   - **HTML File Upload**:
     - Download the verification file
     - Upload to `public/` folder and rebuild
4. After verification:
   - Submit sitemap: `https://verixaservices.com/sitemap.xml`
   - Click "URL Inspection" and submit each key page
   - Set geographic target to Pakistan (Settings → International Targeting)

## 2. Google Business Profile (CRITICAL for Local SEO)

### Steps:
1. Go to https://business.google.com
2. Create business profile:
   - Business name: Verixa Services
   - Category: Software Company, IT Services, AI Development
   - Address: Your Karachi office address
   - Service areas: Karachi, Pakistan, USA, Canada, UK, UAE, Saudi Arabia
   - Phone: Your business phone
   - Website: https://verixaservices.com
3. Complete all sections:
   - Add business hours
   - Add services (list all AI/software services)
   - Upload photos (office, team, logo)
   - Add business description (use keywords!)
4. Post weekly updates about projects, services, tech tips

## 3. Bing Webmaster Tools

### Steps:
1. Go to https://www.bing.com/webmasters
2. Sign in with Microsoft account
3. Add site: `verixaservices.com`
4. Verify using Google Search Console (import verification)
5. Submit sitemap: `https://verixaservices.com/sitemap.xml`
